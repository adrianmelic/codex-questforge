# Codex Questforge Starter Audio Pack

This folder contains the bundled starter ambience pack for Codex Questforge.

- `library.json`: approved audio library used by `scripts/audio_library.py`.
- `starter-pack/*.mp3`: curated Suno-generated tracks selected for the first public challenge-ready build.

All included tracks were generated by Adrian Melic with Suno v5.5 for Codex Questforge on 2026-06-08 while using a paid Suno plan intended to grant commercial rights for newly generated outputs. Suno's rights guidance is published at <https://help.suno.com/en/articles/2410177>. Keep subscription and generation proof outside the repo when needed.

Use the pack as an optional ambience layer. The game must not start audio by default; local viewers expose a speaker toggle and remember the player's preference when the browser allows it.

Example:

```powershell
python plugins\questforge\scripts\audio_library.py select `
  --library plugins\questforge\assets\audio\library.json `
  --tag dungeon `
  --tag exploration `
  --intensity 3 `
  --format args
```
